You are two steps away from using PERSPICA!

Step 1: Set up your collector appliance

Most PERSPICA installations require you to set up a collection appliance. This can be set-up in vSphere in less than 5 minutes.

Step 2: Select your integration

Select the ways PERSPICA collects information about your networks and applications. Look here first if you want to see the available integration methods, but remember, you will need to complete Step 1 before you can add them.

When you are finished you can click on the button below to start using the Main Dashboard. If you have a custom integration you may skip this step. Don't worry, you can always get back to this page by clicking Getting started at the top-right menu.
